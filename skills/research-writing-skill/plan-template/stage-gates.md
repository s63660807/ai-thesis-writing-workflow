# 阶段门禁（Stage Gates）

> 用途：把论文全流程拆成阶段，避免跳步导致返工。

## 项目

- 项目名称：
- 最后更新：
- 当前阶段：

## S0 启动与边界

- [ ] 已明确研究问题
- [ ] 已明确输出物
- [ ] 已明确截止日期
- [ ] 已记录写作偏好（无加粗、段间空一行、正文少列表）

## S1 方案与注册

- [ ] 已形成研究方案草稿
- [ ] 已判断是否需要预注册
- [ ] （如适用）已记录注册平台与模板

## S2 文献检索

- [ ] 已记录检索式和检索日期
- [ ] 已完成初步去重
- [ ] 已记录纳排标准
- [ ] 已形成 literature matrix（按主题/方法/局限分类）
- [ ] 已形成 evidence map（证据-论点-引用位置映射）
- [ ] 已建立 `refs/citation-verification.csv`
- [ ] 拟引用来源均已进入引用校验台账

## S2.5 证据综合

- [ ] 引言核心背景段有文献支撑
- [ ] 相关工作每个主题段有证据组合
- [ ] 研究空白至少由综述文献或多篇代表性方法论文支撑
- [ ] 未使用的重要文献已记录原因
- [ ] `论文引用位置` 已与正文段落或 citation slot 对齐
- [ ] 机器初核与用户人工校验状态未混用

## S3 大纲与方法

- [ ] 已完成章节骨架
- [ ] 参考文献/注释已作为最后独立章节写入大纲
- [ ] 已对齐研究类型对应规范
- [ ] 已明确图表与数据计划
- [ ] 方法章节按输入到输出流程组织

## S4 写作与图表

- [ ] 已完成核心章节初稿
- [ ] 已完成关键图表或脚本
- [ ] 已做一轮风格与排版检查
- [ ] 已执行正文污染检查（用户要求/过程说明不得进入正文）
- [ ] 已执行 capability-use audit（能力使用审计）

## D0 实验协议锁定（结果型论文）

- [ ] 已创建 `plan/experiment-protocol.md`
- [ ] 已锁定数据集版本、划分规则、Non-IID 构造、随机种子
- [ ] 已锁定 baseline、指标、硬件/软件环境、日志字段

## D1 方法-实验可追踪

- [ ] 已创建 `plan/review/method-experiment-traceability.md`
- [ ] 引言贡献均映射到方法模块、实验、表图和允许声称

## D2 表图数据契约

- [ ] 已创建 `tables/table-schema.md`
- [ ] 已创建 `figures/data-manifest.md`
- [ ] 数据文件位于 `figures/data/`，mock/synthetic 数据已明确标注

## D3 结果族回填

- [ ] 主实验有原始日志、聚合规则、表格、图脚本、正文解释
- [ ] 效率实验有原始日志、聚合规则、表格、图脚本、正文解释
- [ ] 消融实验有原始日志、聚合规则、表格、图脚本、正文解释
- [ ] 泛化实验有原始日志、聚合规则、表格、图脚本、正文解释
- [ ] XAI 分析有解释稳定性、开销和领域一致性证据

## D4 结果章节去污染

- [ ] Results/Discussion 正文不含实验目的、表位、图位、回填模板、讨论提示等过程性说明

## D5 同行评审通过

- [ ] 已创建 `plan/review/results-peer-review.md`
- [ ] 已检查复现性、公平 baseline、声称支撑、消融充分性和表图有效性

## S5 质量与合规

- [ ] 已做报告规范逐项核查
- [ ] 已做引用一致性检查
- [ ] 已核对正文引用、最终参考文献章和 `refs/citation-verification.csv`
- [ ] 用户标记虚构、不可用或不匹配的引用已删除或替换
- [ ] 已检查伦理/利益冲突/AI使用披露要求

## S6 投稿包准备

- [ ] 已准备 cover letter
- [ ] 已准备作者贡献声明
- [ ] 已准备数据共享声明（如适用）
- [ ] 已完成期刊格式终检

## S7 返修与归档

- [ ] 已建立审稿意见响应矩阵
- [ ] 已完成版本变更记录
- [ ] 已归档关键复现材料
