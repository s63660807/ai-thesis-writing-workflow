# 法学写作专项

本技能针对法学论文的写作特点，提供专项指导。

## 一、法学论文特点

### 与其他学科的区别

| 方面 | 法学论文 | 一般文科 |
|------|----------|----------|
| 核心 | 法律论证、规范分析 | 论证逻辑 |
| 引用 | 法律文件、司法案例、学说 | 学术文献 |
| 结构 | 问题导向、规范分析 | 灵活多样 |
| 语言 | 严谨精确、术语规范 | 可适度文学化 |

### 常见法学论文类型

- **理论研究**：法学基础理论、法哲学
- **规范分析**：法条解释、制度分析
- **案例研究**：典型案例分析、判例评析
- **比较法研究**：域外法律制度比较
- **实证研究**：法律实施效果分析
- **立法研究**：立法建议、制度完善

## 二、法学论文结构

### 常见结构模式

**问题导向结构（最常用）**
```
一、问题的提出
二、现行法律规定及其不足
三、域外经验与比较借鉴
四、完善建议
五、结语
```

**规范分析结构**
```
一、引言
二、[制度/概念]的界定
三、[制度/概念]的构成要件
四、[制度/概念]的法律效果
五、[制度/概念]的适用问题
六、结论
```

**案例研究结构**
```
一、案情简介与争议焦点
二、裁判理由分析
三、学理评析
四、制度反思与建议
```

**比较法结构**
```
一、问题的提出
二、我国现行规定
三、域外立法例（德国法、日本法、英美法）
四、比较与借鉴
五、完善建议
```

## 三、法学引用规范

### 基于《法学引注手册》（2020年版）

#### 著作引用
```
王利明：《民法总则研究》，中国人民大学出版社2018年版，第156页。
```

#### 期刊论文引用
```
张明楷：《刑法学中的概念使用与概念界定》，载《法学研究》2021年第3期，第75页。
```

#### 法律文件引用
```
《中华人民共和国民法典》第1165条第1款。
```

#### 司法案例引用
```
张某诉李某合同纠纷案，最高人民法院（2020）最高法民终123号民事判决书。
```

#### 司法解释引用
```
《最高人民法院关于适用〈中华人民共和国民法典〉合同编通则若干问题的解释》（法释〔2023〕13号）第5条。
```

### 外文文献引用

#### 英文著作
```
H.L.A. Hart, The Concept of Law (Oxford University Press 2012), p. 89.
```

#### 英文期刊

## 四、法学写作规范

### 语言风格

**准确性**
- ❌ 这个法律规定不太合理
- ✅ 该条款在适用中存在以下问题

**规范性**
- ❌ 法院判错了
- ✅ 该判决在法律适用上存在值得商榷之处

**客观性**
- ❌ 我认为应该修改法律
- ✅ 基于上述分析，现行规定有待完善

### 常用法学表达

**引出问题**
- 实践中存在...的问题
- 学界对此存在争议
- 现行法对此规定不明

**分析论证**
- 从文义解释角度
- 从体系解释角度
- 从目的解释角度
- 从历史解释角度

**比较借鉴**
- 域外立法例表明...
- 比较法上的经验显示...

**得出结论**
- 综上所述
- 基于以上分析
- 由此可见

## 五、法学研究方法

### 规范分析法

1. 确定待分析的法律规范
2. 文义解释：字面含义
3. 体系解释：与其他条文的关系
4. 目的解释：立法目的
5. 历史解释：立法背景
6. 比较解释：域外规定

### 案例分析法

1. 案情梳理：事实认定
2. 争议焦点：法律问题提炼
3. 裁判分析：法院论证逻辑
4. 学理评析：理论支撑或批评
5. 制度反思：规则完善建议
