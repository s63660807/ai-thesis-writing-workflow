# 论文结构模板

根据论文类型自动推荐章节结构, 用户只需确认或微调。

## 本科毕业论文

```
1. 摘要
2. 绪论
   - 研究背景与意义
   - 国内外研究现状
   - 研究内容与方法
   - 论文结构安排
3. 理论基础/文献综述
4. 研究设计与方法
5. 实验与分析/实证研究
6. 结论与展望
7. 参考文献与注释（独立章）
8. 致谢
```

**字数要求**: 8000-15000字

## 博士学位论文

```
1. 摘要
2. 绪论
   - 研究背景与问题提出
   - 研究目的与意义
   - 研究方法与技术路线
   - 论文创新点
   - 论文结构
3. 文献综述
   - 理论基础
   - 研究现状与发展趋势
   - 现有研究的不足
4. 研究方法
   - 研究框架
   - 研究设计
   - 数据来源与处理
   - 分析方法
5. 研究结果
   - 描述性分析
   - 主要发现
6. 讨论
   - 理论贡献
   - 实践意义
   - 局限性
7. 结论与未来研究方向
8. 参考文献与注释（独立章）
9. 附录
10. 致谢
```

**字数要求**: 50000-100000字

## SCI论文

```
1. Abstract
2. Keywords
3. Introduction
4. Materials and Methods
5. Results
6. Discussion
7. Conclusion
8. References / Notes (independent final chapter)
```

**字数要求**: 3000-8000 words (取决于期刊)

## 开题报告

```
1. 研究背景与意义
2. 国内外研究现状
3. 研究目标与内容
4. 研究方法与技术路线
5. 预期成果与创新点
6. 研究计划
7. 参考文献与注释（独立章）
```

**字数要求**: 3000-5000字

## 使用说明

1. 用户确认论文类型后, 自动加载对应结构
2. 展示结构时, 询问用户是否需要调整
3. 常见调整:
   - 增加案例研究章节
   - 合并/拆分某些章节
   - 调整章节名称
4. 记录最终确认的结构到 plan/outline.md
